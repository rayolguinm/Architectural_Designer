C a s a J u a n a

Croma Arquitectura · Social House Ampliation · 1,500 SqFt

Casa Juana improves the communication between interior spaces and the sun's position. Two new volumes are added to the existing state — a studio, terrace, garage, and a main bedroom. With only 4.5 meters in front and two built levels, the project is a study in the optimal dimensions for the basic spaces every house should have. Extraordinary view of the Popocatépetl Volcano.
